# 💰 HarclikTakibi

HarclikTakibi is an Android application developed with Java that helps users manage their pocket money and expenses. Users can register, log in, add income and expenses, view statistics with charts, and track their remaining balance.

## 📱 Features

- User Registration and Login
- Profile Management
- Add Pocket Money
- Add Expenses
- Remaining Balance Calculation
- Expense Visualization with Pie Charts
- Real-Time Dashboard Updates
- Currency Exchange Rates (USD, EUR, GBP)
- User-Specific Data Management
- Logout Confirmation Dialog

## 🛠 Technologies Used

- Java
- Android Studio
- PHP
- MySQL
- Retrofit
- MPAndroidChart
- XAMPP
- JSON

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/rabiatuncekin/HarclikTakipApp.git
```

2. Open the project in Android Studio.

3. Start Apache and MySQL using XAMPP.

4. Import the MySQL database.

5. Place PHP files inside:

```
xampp/htdocs/HarclikTakipAPI
```

6. Run the Android application.
